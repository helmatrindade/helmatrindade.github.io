Meus Projetos

Trybe Wallet

Uma carteira de controle de gastos com conversor de moedas, desenvolvida em JavaScript com Redux e testes unitários.

Trybe Futebol Clube

No projeto Trybe Futebol Clube, desenvolveu-se um backend dockerizado usando TypeScript e Sequelize para modelagem de dados. A API, alinhada com as regras de negócio, requer autenticação por token para adição de partidas. Estabeleceu-se um relacionamento entre as tabelas "teams" e "matches". O banco de dados MySQL está em um container Docker, gerenciado pelo docker-compose. As Dockerfiles nas raízes do front-end e back-end garantem uma inicialização eficiente do projeto.

API de Blogs

O projeto API de Blogs é um backend desenvolvido em JavaScript, utilizando Node.js e Sequelize para operações CRUD de posts. Implementa autenticação de usuário para postagem, com rotas como POST /login. Inclui categorias para posts, estabelecendo relações entre posts-categorias e categorias-posts. Migrations e modelos foram criados para users, categories, blog_posts e posts_categories, garantindo uma estrutura sólida para a gestão de conteúdo do blog.
